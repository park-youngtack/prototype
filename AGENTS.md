# 언어 관련 기본 규칙 (강제)

- 에이전트가 생성하는 **모든 답변, 설명, 보고, 상태 업데이트는 반드시 한국어**로 한다.
- 사용자에게 되묻거나 질문할 때도 **항상 한국어로 질문**한다.
- 코드 내부의 주석을 작성하거나 변경할 때도 **항상 한국어를 사용**한다.
- 영어 또는 다른 언어로 답변할 필요가 있을 경우에도,  
  **먼저 한국어로 설명한 뒤**, 필요한 부분만 영어 단어를 보조적으로 사용할 수 있다.
- 사용자의 요청에 영어가 포함되어 있어도,  
  **출력 결과와 대화는 한국어를 우선**으로 한다.

# 주석 처리 규칙

- 기존 한국어 주석은 언어와 의미를 유지하고, 꼭 필요한 경우에만 최소한으로 수정한다.
- 한국어 주석을 영어/중국어/일본어/한자 등 다른 언어로 바꾸지 않는다.
- 새 주석이 필요할 때는 반드시 **한국어로만 작성한다**.
- 코드 리팩터링 시, 기존 주석은 가급적 수정 없이 유지한다.

# 일반 행동 지침

- 가능하면 단계적으로, 한국어로 진행 상황을 간단히 보고한다.
- 사용자가 명시적으로 영어를 요구하지 않는 한,  
  **대화·작업·지시·해설 모든 출력은 한국어가 기본값**이다.
- 모든 파일은 **처음부터 UTF-8로 읽는다**. (판단 없이 강제)
- 작업 시작 전에 **루트의 README.md를 UTF-8로 먼저 읽고** 그 규칙과 구조를 따른다.

## Skills
스킬은 `SKILL.md` 파일에 저장된 로컬 지침 묶음이다. 아래는 이 세션에서 사용할 수 있는 스킬 목록이며, 각 항목에는 이름, 설명, 파일 경로가 포함된다. 특정 스킬을 사용할 때는 해당 경로의 내용을 열어 전체 지침을 확인한다.
### 사용 가능한 스킬
- skill-creator: 효과적인 스킬을 만드는 가이드. Codex의 기능을 확장하는 새 스킬을 만들거나(또는 기존 스킬을 업데이트할 때) 사용한다. 전문 지식, 워크플로우, 도구 통합이 필요한 경우에 해당한다. (파일: C:/Users/박영택/.codex/skills/.system/skill-creator/SKILL.md)
- skill-installer: 큐레이션된 목록 또는 GitHub 저장소 경로에서 스킬을 $CODEX_HOME/skills에 설치한다. 설치 가능한 스킬 목록 요청, 큐레이션 스킬 설치, 또는 다른 저장소(비공개 포함) 설치 요청 시 사용한다. (파일: C:/Users/박영택/.codex/skills/.system/skill-installer/SKILL.md)
### 스킬 사용 방법
- 발견: 위 목록이 이 세션에서 사용 가능한 스킬 목록이다(이름 + 설명 + 파일 경로). 스킬 본문은 각 경로에 있다.
- 트리거 규칙: 사용자가 스킬 이름을 지정하거나(`$SkillName` 또는 일반 텍스트), 작업이 스킬 설명에 명확히 부합하면 그 턴에서 해당 스킬을 사용한다. 여러 스킬이 언급되면 모두 사용한다. 재언급이 없다면 다음 턴으로 자동 승계하지 않는다.
- 누락/차단: 지정된 스킬이 목록에 없거나 경로를 읽을 수 없으면, 간단히 알리고 가능한 최선의 대안으로 진행한다.
- 스킬 사용 절차(점진적 공개):
  1) 사용할 스킬을 결정한 뒤 해당 `SKILL.md`를 연다. 워크플로우를 따를 만큼만 읽는다.
  2) `SKILL.md`가 `references/` 같은 추가 폴더를 가리키면, 요청에 필요한 파일만 로드하고 전체를 한꺼번에 로드하지 않는다.
  3) `scripts/`가 있으면 큰 코드를 재작성하기보다 실행/패치하는 방식을 우선한다.
  4) `assets/`나 템플릿이 있으면 새로 만들기보다 재사용한다.
- 조정 및 순서:
  - 여러 스킬이 적용되면 요청을 최소 범위로 덮는 스킬만 선택하고 사용 순서를 알린다.
  - 사용 중인 스킬과 이유를 한 줄로 알린다. 명백한 스킬을 생략했다면 그 이유를 말한다.
- 컨텍스트 위생:
  - 컨텍스트를 작게 유지한다. 긴 내용은 요약하고, 필요한 파일만 로드한다.
  - 깊은 참조 추적을 피한다. 막히지 않는 한 `SKILL.md`에서 직접 링크한 파일만 연다.
  - 프레임워크/프로바이더/도메인 변형이 있으면 관련 파일만 선택하고 그 선택을 알린다.
- 안전 및 대안: 스킬 적용이 어렵다면(파일 누락/지침 불명확 등) 문제를 알리고, 차선의 방법으로 계속 진행한다.
